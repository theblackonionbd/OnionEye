How It Works
1
Load Wazuh CSV Exports
Export alert CSV files from Wazuh Manager (any severity level — 6 through 15). Drag and drop multiple files simultaneously. Or connect directly via the Wazuh API for live data ingestion.
2
Select Time Range & Filters
Apply quick presets (1H, 6H, 1D, 1W, 1M) or custom datetime ranges. Filter by minimum alert level, specific IP addresses, or keywords. Combine filters for surgical precision.
3
Run Analysis Engine
The OnionEye detection engine scans every log entry against 35+ attack signatures, extracts payloads, maps MITRE techniques, identifies C2 domains, and builds complete attacker profiles in seconds.
4
Investigate Across 18 Pages
Navigate through Overview, Charts, Geo Map, Attacker Profiles, Attack Types, Kill Chain, MITRE Matrix, Database Forensics, Credentials, Compromise Evidence, FIM, Payloads, Network Topology, and more.
5
Export Professional Reports
Generate an 8-page dark-themed PDF forensic report with cover page, full statistics, attacker profiles, database commands, payload list, and complete event timeline. Or export filtered data as CSV.
